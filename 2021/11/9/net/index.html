网络编程学习记录

2021年11月09日 810 字 大概 3 分钟

网络编程的学习记录

InetAddress类

方便对IP地址的获取和操作
常用方法：
　　static InetAddress getByName(String host)：确定主机名称的IP地址，主机名称可以是机器名称，也可以是IP地址
　　String getHostName()：获取此IP地址的主机名
　　String getHostAddress()：返回文本显示中的IP地址字符串

UDP收发数据

使用DatagramSocket类作为基于UDP协议的Socket
发送数据的步骤：

  1. 创建发送端的Socket对象　DatagramSocket()
  2. 创建数据，并把数据打包　DatagramPacket(byte[] buf, int length, InetAddress address, int port)
  3. 调用DatagramSocket对象的方法发送数据　void send(DatagramPacket dp)
  4. 关闭发送端　void close()
    demo：

接收数据的步骤：

  1. 创建接收端的Socket对象　DatagramSocket(int port)
  2. 创建一个数据包，用于接收数据　DatagramPacket(byte[] buf, int length)
  3. 调用DatagramSocket对象的方法接收数据　void receive(DatagramPacket dp)
  4. 解析数据包，并把数据在控制台显示　byte[] getData() //返回数据缓冲区，注意，初始化的byte[]数组多余字符会设置为空，记得转换成String的时候用dp.getLength()控制输出长度
  5. 关闭接收端　　void close()
    demo：

TCP收发数据

客户端使用Socket类，服务器端使用ServerSocket类，TCP是可靠连接，必须先启动服务器端接收，再启动客户端发送
发送数据的步骤：

  1. 创建客户端的Socket对象　Socket(InetAddress address, int port)
  2. 获取输出流，写数据　OutputStream getOutputStream() // 返回此套接字的输出流
  3. 释放资源　void close()
    demo：

接收数据的步骤：

  1. 创建服务器端的Socket对象　ServerSocket(int port)
  2. 获取输入流，读数据，并把数据显示在控制台　Socket accept() // 侦听此套接字并接受它　InputStream getInputStream() // 返回此套接字的输入流
    注意：接收数据时如果循环接收，需要一个结束标记符代表数据已全部接收，此时有两种解决方法，1.自定义结束标记符，约定如接收到此标记则结束接收；2.发送端使用shutdownOutput()方法，关闭套接字的输出流，从而使TCP终止一次序列
  3. 释放资源　void close()
